# TechPressBooks Content Strategy

This directory contains the first 5 blog posts implementing the content strategy for TechPressBooks.

## Content Strategy Overview

**Positioning:** Practical, production-focused technical education
**Goal:** Attract the right audience and convert them to book readers

### Content Pillars
1. System Design (PRIMARY)
2. Detection Engineering / Security
3. Production Engineering Mindset
4. AI / LLM Engineering (optional)
5. Engineering Thinking (meta)

### Content Types
1. "Why things fail" (high engagement)
2. "How it actually works" (authority)
3. "Practical breakdowns" (conversion)
4. "Comparisons / tradeoffs"
5. "Frameworks / mental models"

## Post Structure

Each post follows the conversion pattern:

1. **Hook** - Problem statement that resonates
2. **Explanation** - Clear breakdown of concepts
3. **Real-world framing** - Production reality and tradeoffs
4. **Soft CTA** - Link to related book

## Internal Linking Strategy

- Link to 1-2 related blog posts
- Link to 1 related book
- Use contextual CTAs that feel natural
